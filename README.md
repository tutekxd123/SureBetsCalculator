# SureBetsCalculator

SureBets is a program written in C++/QT that could calc stakes for surebets with lay option.
The calculator was inspired by [BetBurger](https://www.betburger.com/)


# Surebetting / Arbitrage Betting
Arbitrage betting (or "sure betting") is a strategy where you cover all possible outcomes of a sporting event across different bookmakers to lock in a guaranteed profit, regardless of the result. It exploits price discrepancies caused by differing odds models and margins


## Features
Stake Calculation Scenarios:
- Multiple Back Odds
- Multiple Back Odds + 1x Lay Odd

Supports Two Currencies: PLN,EUR
Automatic EUR/PLN exchange rate download from API: [LINK NBP](https://api.nbp.pl/api/exchangerates/rates/a/EUR/?format=JSON)
